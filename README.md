# 🥤 團購點餐去 (Group Order System)

這是一個專為辦公室/團體設計的「團購點餐小工具」，解決訂飲料、訂便當的統計麻煩。
採用 **無伺服器 (Serverless)** 架構，直接開啟網頁即可使用，所有訂單資料透過 **Firebase** 雲端即時同步！

## ✨ 主要功能

### 1. 📋 雙模式點餐
- **訂飲料 Mode**：
  - 完整支援 🥤 甜度 / 🧊 冰塊 / 🍬 加料 / 💰 價格 / 📝 備註。
  - 選項可透過「設定」功能動態調整。
- **訂便當 Mode**：
  - 支援 🍱 主餐 / 🍚 飯量 (正常/少飯/半飯/不要飯/換菜) / 🥬 配菜 / 💰 價格。

### 2. ⚡ 即時雲端同步 (Realtime Sync)
- 只要有人新增、刪除或修改訂單，**所有人的螢幕會自動更新**。
- 不用手動重新整理網頁，隨時掌握最新團購進度。

### 3. 📷 互動式菜單 (Menu Viewer)
- **多種上傳方式**：支援按鈕上傳，或直接 **Ctrl+V 貼上**、**拖拉圖片** 上傳。
- **支援 GIF 動圖**：上傳動態菜單也會動喔！
- **智慧瀏覽**：支援滑鼠滾輪 **放大縮小**、滑鼠拖曳 **移動檢視**。
- **視窗置中**：菜單視窗固定於畫面中央，方便邊看邊點。

### 4. ⚙️ 動態選項設定
- 點擊左下角「⚙️ 設定」按鈕。
- 可新增、修改、刪除 **甜度** 與 **冰塊** 的預設選項。
- 設定變更後，所有人的下拉選單也會同步更新。

### 5. 📊 自動統計 & 複製
- **智慧群組**：自動將相同的品項合併顯示 (例如：`雞排飯 x3`, `珍奶 (半糖/少冰) x5`)。
- **財務統計**：即時計算總份數與總金額。
- **一鍵複製**：點擊「複製統計結果」按鈕，即可將明細貼到 Line 或 Slack 群組回報。

## 🚀這用方式

1. 下載本專案資料夾。
2. 雙擊開啟 `index.html` (建議使用 Chrome / Edge 瀏覽器)。
3. 若要多人連線，請確保大家的網頁連線到同一個 Firebase 資料庫 (預設已配置)。

## 🛠️ 技術規格

- **Frontend**: HTML5, CSS3 (Dark Mode Theme), jQuery
- **Backend/Database**: Google Firebase Realtime Database
- **Icons**: SVG / Emoji
- **Plugins**: SweetAlert (美化彈窗), jQuery UI
